**プロジェクト**（3D のオレンジ層）は **[開発.md](./開発.md)** の「プロジェクト」表と **`PROJECT_NODES`**（下記）で揃える。コード置き場の入口は通常 **`README.md`**（クリックで開く文書の正とする）。

**`組織図_ブルースター版.html`**（Canvas）も **`DEPTS` の `file`**（実体は `部署/◯◯.md`）と **`CEO_PATH`**（`ARIS/profile.md`）を上表・下記と揃える。役割ノードで別ファイルへ飛ばす場合は HTML 内の **`ROLE_PATH_OVERRIDE`** を **`PROJECT_NODES`** と整合させる。

リポジトリルートからの相対パス（`href`）でリンクする。`hint` は球体上の補助ラベル（日本語）。

  { name: "Boss", path: "ガバナンス/board.md", hint: "議長" },
  { name: "ARIS", path: "ARIS/profile.md", hint: "最高経営責任者" },
  { name: "全社執行", path: "部署/部署一覧.md", hint: "執行" },
];

const DEPTS = [
