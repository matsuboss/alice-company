---

## フォーマット

```
### [DECISION-XXX] タイトル
- **日付**: YYYY-MM-DD
- **決定者**: Boss / ARIS / 共同
- **背景**: どんな状況がこの決定を促したか
- **検討した選択肢**: どんな代替案があったか
- **決定内容**: 何を決めたか
- **根拠**: なぜこの選択肢か
- **影響**: 結果として何が変わるか
- **ステータス**: 有効 / 上書き済み / 撤回済み
```
