# 意思決定ログ

> *真実の記録。すべての重要な決定はここに記録される。*
> *ARIS管理。Bossはいつでも確認可能。*

---

## フォーマット

```
### [DECISION-XXX] タイトル
- **日付**: YYYY-MM-DD
- **決定者**: Boss / ARIS / 共同
- **背景**: どんな状況がこの決定を促したか
- **検討した選択肢**: どんな代替案があったか
- **決定内容**: 何を決めたか
- **根拠**: なぜこの選択肢か
- **影響**: 結果として何が変わるか
- **ステータス**: 有効 / 上書き済み / 撤回済み
```

### [DECISION-003] 戦略数値目標の指標を企業価値に訂正

- **決定者**: Boss
- **背景**: DECISION-002 の文書化時、ARISが「1億円・300億円」を**売上高**と推測した。Bossの意図は**企業価値**であった。

- **根拠**: Bossの明確な指示。数値目標の**指標の種類**は推測で置き換えない。
- **影響**: 全戦略ドキュメント・ロードマップ・ARISプロフィールの記述を企業価値ベースに統一。OKRや財務モデルは「価値ドライバー」全体を扱う。
- **ステータス**: 有効

### [DECISION-005] 企業価値算定ポリシー正式版（v1.0）の承認
- **日付**: 2026-04-11
- **決定者**: Boss
- **背景**: 草案のチェックリスト項目を確定し、運用に落とす必要がある。
- **決定内容**: `/finance/enterprise_value_policy.md` を**正式版 v1.0**とし**即時発効**とする。確定した事項は次のとおり。  
  - **会計年度**: 暦年（1月1日〜12月31日）、年度末日 **12月31日**  
  - **投資・第三者評価がない四半期の暫定**: **マルチプル法**を標準とする  
  - **マルチプルの分母（原則）**: **直近12ヶ月売上高（LTM）**（四半期は年率換算可、前提明記）

  - **DCF**: 四半期の単一路線の「正」には使わず、補助分析に任意併記可  
  - **第三者評価**: **年1回必須としない（任意）**  
- **根拠**: Bossの承認（本対話）。
- **影響**: 全社は v1.0 に従い暫定評価を行う。DECISION-004 の「草案」状態は完了。
- **ステータス**: 有効
